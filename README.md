# 7kt-Plus
Hosts the recommended scripts and userstyle for my userstyle 7kt-Plus.

7kt-Plus is a userstyle that adds additional css for 7kttube (https://7kt.se/downloads/5/7kttube-2016-redux) to make it more accurate.
